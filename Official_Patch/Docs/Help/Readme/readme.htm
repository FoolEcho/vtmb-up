VAMPIRE®: THE MASQUERADE BLOODLINES™ LISEZMOI

 

Ce fichier lisezmoi contient des informations de dernière minute sur Vampire: The Masquerade Bloodlines.

 

Configuration minimum requise :

 

  • Carte accélératrice 3D - carte graphique 64 Mo et pilotes 100 % compatibles DirectX® 9.0c
  • Processeur Pentium® III 1,2 GHz ou Athlon™ 1,2 GHz ou supérieur
  • 384 Mo de RAM (512 Mo de RAM recommandé)
  • Microsoft® Windows® 98/ME/2000/XP
  • Lecteur CD-ROM 4x (débit de transfert constant de 600 K/sec) et pilotes
  • 3,3 Go d'espace disque non compressé (plus 1,4 Go pour le fichier d'échange Windows®)
  • Carte son 16 bits et pilotes 100 % compatibles DirectX® 9.0
  • Souris, clavier et pilotes 100 % compatibles Windows® 98/ME/2000/XP
  • DirectX® 9.0c (inclus)

 

Chipsets pris en charge par Windows® 98/ME/2000/XP

 

Tous les chipsets ATI® Radeon®

Tous les chipsets NVIDIA® GeForce™

Matrox Parhelia™

 

REMARQUE : ce jeu contient une technologie antipiratage qui peut provoquer des conflits avec certains lecteurs de DVD-ROM/CD-ROM et certains lecteurs virtuels.

 

Dépannage :

 

Direct X 9.0c : le programme d’installation de Vampire: The Masquerade Bloodlines va détecter la version de DirectX® actuellement installée sur votre ordinateur. Si une version plus ancienne de DirectX® est détectée, il vous sera proposé d’installer DirectX® 9.0c. Activision vous recommande d’installer DirectX 9.0c, même si vous pensez avoir déjà cette version. La réinstallation de DirectX 9.0c est sans danger pour votre configuration DirectX.

 

Amélioration des performances : pour de meilleures performances, veuillez vous assurer que vous possédez les pilotes les plus récents pour votre carte graphique. Pour plus d'informations sur les pilotes les plus récents, contactez votre fabricant de carte graphique. Si la fluidité laisse à désirer, nous vous conseillons de baisser la qualité de la résolution, des images et/ou des détails géométriques. Baisser une de ces options peut permettre d'améliorer les performances du jeu. Ces paramètres d’affichage peuvent être modifiés dans le menu Options du jeu.

Vous devez redémarrer le jeu pour que ces options soient prises en compte.

 

Economiseurs d’écran et d’énergie : les économiseurs d’écran et d’énergie peuvent poser des problèmes pendant le jeu Bloodlines s’ils sont actifs quand vous jouez. Activision recommande de désactiver tous les économiseurs d'écran et d'énergie avant de lancer le jeu.
 
Utilisation de Alt-Tab pendant le jeu : Activision ne prend pas en charge l’utilisation de Alt-Tab pendant le jeu Bloodlines. L’utilisation de Alt-Tab pour changer d’application est susceptible de perturber le programme.

La touche Shift dans Windows 2000(SP2) et XP interrompt le jeu : si vous appuyez sur Shift cinq fois de suite avec Windows® 2000(SP2), le message suivant devrait apparaître :

 

"En pressant la touche Maj 5 fois, vous activez les Touches rémanentes. Avec cette fonctionnalité vous déverrouillez les touches Ctrl, Alt ou Maj. Ceci est utile s’il vous est impossible de maintenir enfoncée plus d'une touche à la fois.


Si vous souhaitez activer cette fonctionnalité cliquez sur OK, sinon cliquez sur Annuler.
<Case à cocher> Désactiver les raccourcis clavier pour cette fonctionnalité d’accessibilité
<OK>                     <ANNULER>         <PARAMETRES>”

 

Pour éviter que cela n’arrive, cliquez sur le bouton Paramètres cité ci-dessus. Quand l'écran des Options d'accessibilité apparaît, cliquez sur le bouton "Paramètres" dans la partie Touches rémanentes de la fenêtre. Sur la fenêtre suivante (qui devrait s'intituler "Paramètres pour les touches rémanentes"), décochez la case "Utiliser le raccourci", puis cliquez sur "OK" et fermez la fenêtre. Le problème ne devrait plus se poser une fois les raccourcis désactivés. Cela peut aussi être fait pour les Touches filtre et les Touches bascule.

Mode sans échec : si la résolution choisie pour le jeu est trop élevée pour votre moniteur, vous pouvez lancer le jeu en mode sans échec. Le lancement du jeu en mode sans échec se fera à une résolution de 800X600, c’est-à-dire la résolution minimale prise en charge par le jeu. Pour lancer le jeu en mode sans échec, allez dans le menu Démarrer>Programmes>Vampire – The Masquerade Bloodlines>Vampire – The Masquerade Bloodlines (mode sans échec).

 

Utilisateurs du chipset audio Nforce™ : certains utilisateurs de la Nforce™ sont susceptibles d'expérimenter des problèmes de son en jouant à Bloodlines. Les utilisateurs qui sont confrontés à ce problème peuvent essayer les deux choses suivantes :

 

- Dans le menu Options, vérifiez que les options « Audio haute qualité » et « Effets environnementaux » sont décochées.

 

- Vérifiez que votre accélération DirectSound® est réglée sur « Basique ». Vous pouvez procéder comme suit :

               

1.) Cliquez sur le bouton « Démarrer » de Windows®.

2.) Cliquez sur l’option « Exécuter » dans le menu Démarrer.

3.) Tapez « dxdiag » sans les guillemets dans la fenêtre de dialogue Exécuter.

4.) Une fois que l’outil de diagnostique DirectX® a terminé, cliquez sur l'onglet « "Son ».

5.) Mettez le « Niveau d’accélération matérielle » sur « Basique ».

Changements de dernière minute :

 

Ci-dessous une liste des changements de dernière minute dans le jeu ---

 

Horloge d’auto-renouvellement – L’horloge d’auto-renouvellement, qui est disponible dans les options de jeu, change la manière dont l'horloge des disciplines fonctionne. Avec l'auto-renouvellement activé, les disciplines passives sont automatiquement réactivées lorsque leur effet prend fin, et vous pouvez activer et désactiver une discipline par un clic droit. C’est une option avancée utile, car il est très facile de se vider complètement de son sang si vous ne faites pas attention. Souvenez-vous que F8 désactive toute discipline active.

 

Montrer dégâts chiffres – Si vous désactivez cette option, vous ne pourrez plus voir les dégâts en chiffres quand un personnage subit des dégâts.

 

Touches de raccourci – Appuyer sur "K" (par défaut) affiche le menu des raccourcis clavier. Dans ce menu, vous pouvez attribuer des fonctions à toutes les touches alphanumériques en haut du clavier. L’attribution de raccourcis clavier ne fonctionne pas pour le pavé numérique. Pour attribuer une action à une touche, choisissez une action dans le menu déroulant près du numéro de la touche à laquelle vous voulez attribuer une action. Pour les armes et les disciplines, choisissez une arme ou une discipline dans le menu déroulant du bas. De plus, vous pouvez attribuer des objets à utiliser à des raccourcis. Quand vous appuyez sur un raccourci clavier, vous utilisez automatiquement l’arme ou l'objet attribué. Veuillez remarquer que les disciplines ne seront pas automatiquement activées par les raccourcis clavier. Vous pourrez modifier la discipline attribuée via le raccourci clavier, mais vous devrez l’activer par un clic droit de la souris (par défaut).

Combat et dégâts :

 

Voici une explication détaillée du déroulement d'un combat et de la manière dont les dégâts sont calculés dans le jeu.

 

Chaque arme du jeu dispose d’attributs spécifiques. Ces attributs sont les suivants :

 

Seuil de combat à distance - C’est le niveau minimum auquel le personnage joueur peut utiliser efficacement son arme. Le personnage joueur peut utiliser des armes si son aptitude est inférieure au seuil. Par contre, cela signifie qu’il infligera moins de dégâts. A l’inverse, si l’aptitude de combat d’un personnage joueur est supérieure à ce minimum, il causera davantage de dégâts.

 

Potentiel de dégâts - Le potentiel de dégâts est le maximum de dégâts que votre personnage peut infliger avec son arme à son niveau actuel. Les dégâts effectifs peuvent varier selon la capacité de l'adversaire à se protéger des balles (par exemple, si les dégâts sont d’origine surnaturelle ou si l’adversaire possède un gilet pare-balles).

 

Létalité – C’est la quantité de dégâts que peut infliger un joueur avec son arme par dé.

 

Ajustement d’aptitude – C’est la différence entre le seuil de l’arme et l’aptitude de combat du personnage. Elle peut être positive ou négative.

 

Dégâts de base - C’est le minimum de dégâts qu’une arme peut infliger.

 

A chaque coup porté, les dommages sont calculés de la façon suivante :

 

Potentiel de dégâts = Dégâts de base X (Létalité + Ajustement d’aptitude) :

 

Par exemple, si le personnage joueur qui a une aptitude de Mêlée de 5 utilise un couteau qui a un seuil de combat de Mêlée de 4, une létalité de 7 et des dégâts de base de 3, alors le potentiel de dégâts est de :

24 = 3 X (7+1).

 

Par contre, si l’aptitude de combat de Mêlée du joueur est de 3, alors le potentiel de dégâts est de 18 = 3 X (7-1)

 

Il est important de comprendre qu'un joueur n'est mortellement efficace que lorsque son aptitude de combat est supérieure au seuil de son arme.

 

Remarque : la Défense et l’Absorption de la cible peuvent absorber des dégâts et diminuer la Létalité. Par exemple, si une goule a une armure lourde, qui a une Défense de 2, il est possible que 2 dés de Létalité soient décomptés au jet du joueur. Dans le premier exemple, si le personnage joueur frappe une goule avec un couteau et que la goule fait deux jets de Défense, alors le potentiel de dégâts sera de :

 

18 = 3 X (7+1-2)

 

Donc, quand un joueur atteint un haut potentiel de dégâts avec une arme, les dégâts peuvent être réduits si la cible a une aptitude de Défense ou d’Absorption élevée.

 

Le type d’arme et le défenseur jouent un rôle important dans le type de jet d’Absorption utilisé.

Pour se défendre efficacement contre une attaque ou l’absorber, un jet est effectué par l’ordinateur. Quand il y a une chance d'absorber ou de se défendre, un jet est effectué sur un dé 10 (dé à 10 faces). Si le chiffre est supérieur ou égal à 7, le défenseur parvient à annuler un dé.

 

Le combat peut donc être résumé de la façon suivante : il est beaucoup plus difficile de se défendre contre une arme à létalité élevée, et les armes à dégâts de base élevés nécessitent une certaine adresse.
